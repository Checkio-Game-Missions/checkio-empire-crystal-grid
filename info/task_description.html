First checking phase (link) is not enough and we should improve this process. Let's check random slices instead rows. Our crystal contains two atoms - "X" (Xenatom) and "Z" (Zorium). These atoms should alternate each other in rows and columns.

You are given a slice from crystal lattice as a grid (2D array) of letters "X" and "Z". The good grid should have the periodic arrangement by horizontals and verticals. If any atoms are repeated, then this crystal is broken. For example: 

[["X", "Z"],
 ["Z", "X"]] is good

[["X", "Z", "X"],
 ["Z", "X", "Z"],
 ["X", "Z", "X"]] is good

[["X", "Z", "X"],
 ["Z", "Z", "Z"],
 ["X", "Z", "X"]] is bad

Input: Atom grid as a list of list with strings.

Output: Is a grid good or not as a boolean.

Example:

How it is used: In this mission we will use more complex structure as list of list. This structure often use as tables or grids or land maps.

Precondition:
1 < len(grid) ≤ 10
1 < len(grid[0]) ≤ 10
all(len(row) == len(grid[0]) for row in grid)
all(all(ch in "XZ" for ch in line) for line in grid)
